## Smart Access Gauge for Industrial Pressure Monitoring using Hall Effect Sensor and ESP32

## Overview

This project was developed during my internship at WIKA Instruments.
The objective was to convert a conventional pressure gauge into a smart digital monitoring device using Hall Effect sensing and embedded visualization.
The system detects pointer movement using a Hall Effect sensor and processes the measurements through an ESP32-based embedded platform

## Features
- Real-time pressure monitoring
- Hall Effect based sensing
- ESP32 embedded processing
- Embedded visualization
- Industrial pressure monitoring
- Analog gauge digitization
- 
## Hardware Components
- ESP32
- Hall Effect Sensor
- Permanent Magnet
- TFT Display
- Industrial Pressure Gauge
- Custom PCB

## Working Principle
1. Magnet attached to pressure gauge pointer.
2. Hall sensor detects magnetic field variation.
3. ESP32 processes sensor readings.
4. Calibration converts readings to pressure values.
5. Values displayed in real time.
